# 自动输入功能使用说明

## 功能概述

Windows 客户端现在支持两种文本同步模式：

### 1. 📋 剪贴板模式（默认）
- **工作方式**：收到 Android 端的文本后，仅写入 Windows 剪贴板
- **适用场景**：需要手动控制粘贴时机，或在不希望自动输入的场景下使用
- **使用方法**：收到文本后，手动按 `Ctrl+V` 粘贴到目标位置

### 2. ⌨️ 自动输入模式
- **工作方式**：收到 Android 端的文本后，自动输入到当前焦点窗口
- **适用场景**：光标已在输入框中，希望直接输入文本
- **使用方法**：
  1. 启用"自动输入模式"开关
  2. 将光标放在目标输入框（如记事本、Word、浏览器等）
  3. Android 端发送文本后，会自动输入到该输入框

## 自动输入模式的两种方式

当启用自动输入模式后，可以选择以下两种输入方式：

### 方式一：粘贴方式（推荐）✅
- **工作原理**：先写入剪贴板，然后模拟 `Ctrl+V` 粘贴
- **优点**：
  - 速度快，适合长文本
  - 准确性高，不会出现字符遗漏
  - 支持特殊字符和 Unicode
- **缺点**：会覆盖当前剪贴板内容

### 方式二：逐字符输入
- **工作原理**：模拟键盘逐个按键输入
- **优点**：
  - 不修改剪贴板内容
  - 更兼容某些特殊应用
- **缺点**：
  - 速度较慢（每个字符间隔 5ms）
  - 长文本可能需要较长时间
  - 可能受输入法影响

## 使用步骤

1. **启动 Windows 服务器**
   ```bash
   cd packages/windows_app
   flutter run -d windows
   ```

2. **配置连接**
   - 查看本机 IP：运行 `ipconfig`，找到 IPv4 地址
   - 在 Android 端设置此 IP 地址
   - 确保手机和 PC 在同一 WiFi 网络

3. **选择输入模式**
   - 打开 Windows 客户端界面
   - 在"输入模式设置"区域选择需要的模式
   - 如选择自动输入，建议启用"使用粘贴方式"

4. **开始使用**
   - 启动 Android 客户端并连接
   - 在 Android 端复制或输入文本
   - 文本会自动同步到 Windows 端

## 注意事项

⚠️ **重要提示**：

1. **焦点窗口**：自动输入模式下，文本会输入到当前有焦点的窗口
   - 确保光标在正确的输入框中
   - 避免在输入过程中切换窗口

2. **权限要求**：
   - 自动输入需要应用程序有前台权限
   - 某些安全软件可能会拦截键盘模拟

3. **输入法影响**：
   - 逐字符输入方式可能受中文输入法影响
   - 建议在英文输入法状态下使用
   - 或使用粘贴方式避免此问题

4. **特殊字符**：
   - 粘贴方式支持所有 Unicode 字符
   - 逐字符输入可能对某些特殊字符支持有限

5. **性能考虑**：
   - 短文本（<100字符）：两种方式都可以
   - 长文本（>100字符）：强烈建议使用粘贴方式

## 常见问题

**Q: 为什么文本没有输入到目标窗口？**
A: 请检查：
- 是否启用了自动输入模式
- 光标是否在目标输入框中
- 目标窗口是否有焦点

**Q: 输入的文本出现乱码？**
A: 建议切换到"粘贴方式"，它对所有字符的支持更好

**Q: 能否在后台运行时自动输入？**
A: 可以，但需要确保目标窗口有焦点。建议最小化服务器窗口，保持目标窗口在前台

**Q: 自动输入会影响剪贴板吗？**
A: 
- 使用"粘贴方式"：会临时修改剪贴板
- 使用"逐字符输入"：不会影响剪贴板

## 技术实现

- **剪贴板操作**：使用 Win32 API (`OpenClipboard`, `SetClipboardData`)
- **键盘模拟**：使用 Win32 API (`SendInput`)
- **虚拟键码**：使用 `VkKeyScanW` 转换字符到虚拟键码
- **修饰键处理**：自动处理 Shift、Ctrl、Alt 组合键

## 示例场景

### 场景 1：快速记录笔记
1. 打开记事本或 OneNote
2. 启用自动输入模式（粘贴方式）
3. 在手机上语音转文字
4. 文本自动出现在记事本中

### 场景 2：填写表单
1. 打开浏览器表单页面
2. 点击第一个输入框
3. 启用自动输入模式
4. 在手机上准备文本
5. 依次发送到各个输入框

### 场景 3：编程代码同步
1. 打开代码编辑器
2. 启用自动输入模式（粘贴方式）
3. 在手机上准备代码片段
4. 自动粘贴到编辑器中
