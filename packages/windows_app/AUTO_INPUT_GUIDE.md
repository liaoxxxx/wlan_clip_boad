# Windows 自动输入功能说明

## ✨ 新功能特性

### 1. 默认开启自动输入模式
- ✅ 应用启动时**自动启用**自动输入模式
- ✅ 无需手动切换，开箱即用
- ✅ 收到 Android 端文本后直接输入到当前焦点窗口

### 2. 实时文本显示
- 📝 界面新增"接收的文本"显示区域
- 👁️ 实时展示从 Android 端接收到的所有文本
- 📋 提供"复制"按钮，可快速复制到剪贴板
- 🎨 紫色边框高亮显示，一目了然

### 3. 智能输入方式
- 🚀 默认使用**粘贴方式**（Ctrl+V）
- ⚡ 速度快，适合长文本
- 🔧 可切换到逐字符输入模式（如需要）

---

## 🎯 使用场景

### 场景 1：在 PC 上输入手机上的文字
1. 在手机上复制一段文字
2. Windows 应用自动接收并显示
3. 在 PC 上点击任意输入框（微信、浏览器、Word 等）
4. 文本**自动输入**到光标位置

### 场景 2：跨设备文本同步
1. 手机上看到有用的信息
2. 复制后自动同步到 PC
3. 在 PC 的任何地方直接使用

### 场景 3：语音转文字输入
1. 手机上使用语音输入
2. 转换后的文字自动发送到 PC
3. PC 端自动输入到目标应用

## ⚙️ 配置选项

### 自动输入模式（默认：✅ 开启）
- **开启**：收到文本后自动输入到当前焦点窗口
- **关闭**：仅写入剪贴板，需手动 Ctrl+V 粘贴

### 粘贴方式（默认：✅ 开启）
- **开启**：使用 Ctrl+V 粘贴（快速，推荐）
- **关闭**：逐字符模拟键盘输入（兼容性好，但速度慢）

## 💡 使用技巧

### 技巧 1：保持窗口最小化
- Windows 应用可以最小化运行
- 不影响自动输入功能
- 节省屏幕空间

### 技巧 2：快速复制文本
- 点击"接收的文本"区域的"复制"按钮
- 可以快速将文本复制到剪贴板
- 方便后续使用

### 技巧 3：查看历史记录
- 在"连接日志"区域查看所有操作记录
- 包含时间戳和详细信息
- 最多保留 50 条记录

### 技巧 4：多设备连接
- 支持多个 Android 设备同时连接
- 每个连接有独立编号
- 日志中会显示连接/断开状态

## 🔧 故障排除

### 问题 1：文本没有自动输入
**可能原因**：
- 没有窗口获得焦点
- 目标应用不支持自动输入

**解决方法**：
1. 确保目标窗口处于活动状态
2. 点击输入框确保光标闪烁
3. 检查是否开启了自动输入模式

### 问题 2：输入速度慢
**可能原因**：
- 使用了逐字符输入模式
- 文本过长

**解决方法**：
1. 确保"使用粘贴方式"已开启
2. 粘贴方式速度更快，推荐使用

### 问题 3：无法连接
**可能原因**：
- IP 地址错误
- 防火墙阻止
- 不在同一网络

**解决方法**：
1. 检查 Windows 端的 IP 地址（运行 `ipconfig`）
2. 确保 Android 端设置正确的 IP
3. 检查防火墙设置，允许端口 8889
4. 确保手机和 PC 在同一 WiFi 网络
