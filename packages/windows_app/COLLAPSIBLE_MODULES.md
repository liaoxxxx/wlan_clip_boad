# 📦 模块折叠功能实现

## ✅ 功能完成

### 功能描述
Windows 应用界面中的所有功能模块现在都支持折叠/展开，用户可以根据需要自定义显示内容。

## 🎨 UI 设计

### 折叠卡片组件

每个模块都是一个可折叠的卡片，包含：

```
┌──────────────────────────────┐
│ 📝 接收的文本          [▼]   │ ← 标题栏（可点击）
├──────────────────────────────┤
│                              │ ← 内容区域（展开时显示）
│  等待接收文本...              │
│                              │
└──────────────────────────────┘
```

### 交互效果

- **点击标题栏**: 切换展开/收起状态
- **图标指示**: 
  - `▼` (expand_more) - 当前已展开，点击可收起
  - `▶` (expand_less) - 当前已收起，点击可展开
- **动画流畅**: 使用 Flutter 内置动画，过渡自然

## 🔧 技术实现

### 1. 状态管理

在 `_WindowsClipboardServerState` 中添加折叠状态变量：

```dart
// 模块折叠状态
bool _serverStatusExpanded = false;    // 服务器状态 - 默认折叠
bool _textInputExpanded = true;        // 接收的文本 - 默认展开 ⭐
bool _inputModeExpanded = false;       // 输入模式 - 默认折叠
bool _windowSettingsExpanded = false;  // 窗口设置 - 默认折叠
bool _instructionsExpanded = false;    // 使用说明 - 默认折叠
bool _logsExpanded = false;            // 连接日志 - 默认折叠
```

### 2. 可复用组件

创建 `_buildCollapsibleCard` 方法：

```dart
Widget _buildCollapsibleCard({
  required String title,           // 标题
  required bool isExpanded,        // 是否展开
  required VoidCallback onToggle,  // 切换回调
  required Widget child,           // 内容
  Color? headerColor,              // 头部背景色
  Color? borderColor,              // 边框颜色
  bool alwaysVisible = false,      // 是否始终可见
})
```

**特性**:
- ✅ 统一的视觉风格
- ✅ 可自定义颜色主题
- ✅ 支持最小模式隐藏
- ✅ 响应式布局

### 3. "接收的文本"特殊处理

由于这是核心功能，采用特殊的内联实现：

```dart
Container(
  decoration: BoxDecoration(...),
  child: Column(
    children: [
      // 标题栏 - 可点击
      InkWell(
        onTap: () {
          setState(() {
            _textInputExpanded = !_textInputExpanded;
          });
        },
        child: Row(
          children: [
            Text('📝 接收的文本'),
            Icon(_textInputExpanded ? Icons.expand_less : Icons.expand_more),
          ],
        ),
      ),
      // 内容区域 - 条件渲染
      if (_textInputExpanded)
        Expanded(child: ...),
    ],
  ),
)
```

**原因**:
- 需要使用 `Flexible` 和 `Expanded` 进行布局
- 保持文本区域的滚动功能
- 确保在紧凑模式下仍然可用
